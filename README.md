# GlycoSnap 🍽 
GlycoSnap is a food recognition app that calculates the **glycemic load** of meals using YOLOv8.  

## 🚀 Features  
✅ Food recognition with YOLOv8  
✅ Portion size estimation (WIP)  
✅ Glycemic Load calculation
